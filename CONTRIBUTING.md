# Contributing

Thank you for your interest in contributing! This repository contains a
complete, MIT-licensed application — contributions that improve self-hosting,
documentation, and genuine feature extensions are welcome.

## How to contribute

1. **Fork** the repository and create a branch from `main`.
2. **Make your changes** — keep them focused and well-tested.
3. **Open a pull request** with a clear description of what changed and why.

## Issue labels

- `good first issue` — well-scoped tasks suitable for first-time contributors.
- `help wanted` — areas where the maintainers would especially welcome help.
- `roadmap` — planned features: check the issue for scope before starting.
- `self-hosting` — deployment and infrastructure questions.

## Code of conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md).
Be kind, be constructive, be collaborative.
